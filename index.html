Extração de Chaves Kindle KFX

Tutorial de Descriptografia e Backup (2026)

🚀 Visão Geral

Este guia ensina como converter arquivos KFX protegidos para formatos abertos como ePub ou PDF usando o Calibre e ferramentas de extração de chaves.

Atenção: Use este procedimento apenas para fins de backup pessoal de livros adquiridos legalmente.

🛠️ Requisitos do Sistema

  • Calibre: Versão v9.0+
  • Kindle para PC: v2.8.0 a v2.8.3
  • Plugin DeDRM: v10.0.18
  • KFX Input: v2.30.0

💻 Passo a Passo

  1. Instale os plugins no Calibre via Preferências > Plugins.
  2. Coloque o KFXArchiver283.exe na pasta do App Kindle.
  3. Execute o comando no CMD para gerar sua chave .k4i.
  4. Importe a chave no plugin DeDRM e converta seus livros.
