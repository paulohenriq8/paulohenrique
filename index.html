Como Converter Livros do Kindle para PDF ou EPUB

Aprenda a remover a proteção (DRM) para ler seus livros em qualquer dispositivo

Guia Atualizado: 10 de Abril de 2026

Aviso de Uso Legal

Este processo destina-se apenas ao uso pessoal em livros que você possui legalmente. O objetivo é garantir que você possa ler seus arquivos em outros leitores ou aplicativos.

Por que este guia é necessário?

Os livros comprados na Amazon vêm com uma trava digital (DRM). Para transformá-los em PDF ou EPUB, precisamos de ferramentas especiais que "desbloqueiam" o arquivo. Este guia explica como usar os plugins gratuitos do Calibre para fazer isso em 2026, compatível com as versões 2.8.0 a 2.8.3 do Kindle para PC.

Passo a Passo Detalhado

01

Preparação das Ferramentas

Você precisará do programa Calibre (v9.3+) instalado. Baixe os dois componentes abaixo clicando nos botões:

Plugin DeDRM (v10.0.18) Plugin KFX Input (v2.30)

Dentro do Calibre, instale-os em: Preferências > Plugins > Carregar plugin de arquivo.

Importante: Reinicie o Calibre após instalar para ativar as funções.
02

Configuração do Desbloqueador

Primeiro, instale a versão oficial do Kindle para PC através do link abaixo:

Baixar Kindle para PC (Oficial)

Depois, copie o arquivo KFXArchiver283.exe para esta pasta exata:

C:\Users\NOME_DE_USUARIO\AppData\Local\Amazon\Kindle\application
03

Baixar os Livros no Computador

Abra o programa Kindle Desktop, faça login e clique duas vezes na capa do livro para garantir que o download seja concluído 100%.

Dica: Se o download falhar ou o arquivo ficar incompleto, a extração da chave no próximo passo não funcionará.
C:\Users\NOME_DE_USUARIO\Documents\My Kindle Content
04

Gerar a Chave de Desbloqueio

Primeiro, crie manualmente uma pasta chamada kfxoutput em C:\Users\NOME_DE_USUARIO\. Sem essa pasta, o comando falhará.

Abra o Prompt de Comando (CMD) e cole o texto abaixo:

C:\Users\NOME_DE_USUARIO\AppData\Local\Amazon\Kindle\application\KFXArchiver283.exe "C:\Users\NOME_DE_USUARIO\Documents\My Kindle Content" "C:\Users\NOME_DE_USUARIO\kfxoutput" kfxoutput.k4i

⚠️ ATENÇÃO:

Troque NOME_DE_USUARIO pelo seu nome real nas 3 ocorrências. Se o CMD mostrar "DRMION decrypted", a chave foi gerada com sucesso.

05

Configurar a Chave no Calibre

Vá em Preferências > Plugins > DeDRM > Personalizar Plugin. Em "Kindle for Mac/PC ebooks", importe o arquivo kfxoutput.k4i.

Importante: Sempre que baixar novos livros, é recomendável limpar a pasta kfxoutput e apagar o .k4i anterior antes de rodar o comando novamente.
06

Importação e Conversão Final

Regra de Ouro: Nunca arraste arquivos .azw da pasta original "My Kindle Content". Você deve arrastar os arquivos gerados que estão na pasta de saída.

  • ✓

    Livros Novos (KFX): Arraste o arquivo kfx-zip diretamente da pasta kfxoutput para a biblioteca do Calibre. Se ele virar "KFX" no Calibre, deu certo.

  • i

    Livros Antigos (KF8/Mobi): Se o livro for um formato antigo, você pode arrastar o arquivo .azw original, desde que a chave .k4i já esteja configurada no plugin.

Após o desbloqueio, use o botão "Converter livros" para gerar seu PDF ou EPUB.

Resolução de Erros

Erro / Sintoma Como Corrigir
"KFX-ZIP" no Calibre Bloqueio ainda presente. Delete o livro do Calibre, limpe a pasta kfxoutput e repita os passos 4 e 5.
"BookOpen error 14" Arquivo corrompido ou antigo. Verifique se o livro foi totalmente baixado no app Kindle antes de extrair.
"Access Violation" (CMD) Conflito de versão. Confirme se está usando exatamente a versão 2.8.3 do Kindle para PC.
